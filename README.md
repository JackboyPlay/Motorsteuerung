# Motorsteuerung
Code, welcher das Steuern eines Motors zulässt.

Konsolenbefehle:
- speed -> erlaubt es die Geschwindigkeit in einem Bereich von 0 - 255 zu ändern.
- mode  -> erlaubt es den Modus in einem Bereich von 0 - 2 zu ändern.
